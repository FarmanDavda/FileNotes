
# FileNotesApp

A simple Java console application that demonstrates file handling using `FileWriter` and `BufferedReader`. This project writes a sample note to a text file and then reads it back from the same file.

## Features

- Write a note to a file (`notes.txt`)
- Read the content of the file and display it on the console
- Exception handling for both write and read operations

## Technologies Used

- Java (Core Java)
- File I/O (FileWriter, FileReader, BufferedReader)

## How to Run

1. Make sure you have Java installed. You can check using:
   ```bash
   java -version
   ```

2. Compile the Java file:
   ```bash
   javac FileNotesapp.java
   ```

3. Run the program:
   ```bash
   java FileNotesapp
   ```

## Output

```
Note Write to the file
Read from file
this is sample note write using FileWriter
```
